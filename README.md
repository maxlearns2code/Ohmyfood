# Ohmyfood

Integrate the mockup in mobile first<br />
Improve the interface with CSS animations

## Prerequisites

- node.js v18
- npm
- nvm
- sass

## Installation

1. Clone the repo
```bash
    git clone https://github.com/maxlearns2code/Ohmyfood.git
```

2. Go to the project directory

```bash
    cd ohmyfood
```

3. use nvm
```bash
    nvm use
```

4. install sass
```bash
    npm i sass
```

5. run sass
```bash
    npm run sass
```
